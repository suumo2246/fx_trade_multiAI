venv\Scripts\activate

gitで管理するもの
fx_trade_multiAI/
├── src/                    # ソースコード
├── config/                 # 設定ファイル
├── tests/                  # テストコード
├── notebooks/              # Jupyter Notebook
├── docs/                   # ドキュメント
├── requirements.txt        # ライブラリ依存関係
├── README.md              # プロジェクト説明
└── .gitignore             # 除外設定

gitで管理しない物
fx_trade_multiAI/
├── venv/                   # 仮想環境
├── data/raw/               # 生データ（大容量）
├── models/*.pkl            # 学習済みモデル（大容量）
├── logs/                   # ログファイル
├── .env                   # 機密情報
└── __pycache__/           # Pythonキャッシュ

再現方法
# 1. リポジトリをクローン
git clone https://github.com/suumo2246/fx_trade_multiAI.git
cd fx_trade_multiAI

# 2. 仮想環境を作成
python -m venv venv
venv\Scripts\activate

# 3. ライブラリをインストール
pip install -r requirements.txt

# 4. 開発開始
